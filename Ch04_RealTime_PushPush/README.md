# 이 프로젝트의 주제
실시간 게임을 만들어보자!
지금까지는 입력이 들어오면 그것에 맞춰서 모든 게임의 정보가 갱신되었지만, 결국은 동기화가 된 입력이 있어야
렌더링 정보가 업데이트가 되고 무조건 이 업데이트에 맞춰서만 그려지는 방식인 것이다.

하지만 이번 프로젝트는 비동기화된 입력을 통해서 정보가 업데이트 되기에,
정보가 업데이트 되기에 입력이 있어야만 다시 그리는 그런 것이 아니라
그리는 것은 입력이 없어도 지속적으로 그려지고, 입력이 생겨 정보가 업데이트 되면 그 때 다르게 그리는 식이다.

또한 실시간으로 그려진다는 장점 덕분에 (즉 그려지는 부분이 굉장히 독립적이란 장점 덕분에)
애니메이션을 확실하게 넣을 수 있어 진정한 실시간 게임을 만들 수 있다.
